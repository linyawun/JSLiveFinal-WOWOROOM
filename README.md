## 說明
此為2022年參加六角學院 JavaScript 直播班的最終作業，版型使用六角提供 html 與 css，只有稍做微調，主要練習 JavaScript 的API串接，功能包含：
### 前台
- 顯示產品列表
- 產品類別篩選
- 加入購物車
- 修改購物車數量
- 刪除單筆與刪除全部購物車
- 表單驗證、送出表單
### 後台
- 顯示訂單列表
- 更改訂單狀態
- 刪除單筆與刪除全部訂單
- 依據訂單金額顯示 c3.js 圖表
